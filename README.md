# 个人博客网站配置

### npm run docs:dev 运行项目
### npm run docs:build 项目打包

## 项目结构
`
docs
├── .vuepress  // 用于存放
│   ├── dist // 打包文件 放在服务器上
│   ├── public // 静态文件
│   ├── config.js // 项目配置文件--包括配置项目标题 侧栏 顶部菜单等
│   └── images // 页面模板
│ 
├── SideBar // 侧栏md文件
│ 
└── README.md // 首页图片及描述性句段

package.json  // 包管理文件

`
